## 步骤3：训练 Critic（当前阶段）

### 预期结果
- 训练后的 Critic 模型保存在 `experiments/debug_3samples/critic_iter0/`
- DPO loss 应该稳定下降
- 训练时间：约10-15分钟（3样本，1 epoch）

### 可能遇到的问题

#### 问题1：训练 loss 不下降
**症状**: loss 保持不变或波动
**可能原因**:
- 学习率过高或过低
- 偏好对质量差（delta 接近0）
- batch_size 不合适

**应对策略**:
```bash
# 如果 loss 不下降，尝试调整参数
--learning_rate 3e-5  # 降低学习率
--batch_size 1        # 减小批次
```

#### 问题2：GPU OOM
**症状**: CUDA out of memory
**可能原因**:
- 单GPU内存不足
- 梯度累积步数过大

**应对策略**:
```bash
# 减小批次和梯度累积
--batch_size 1
# 或使用双GPU分离
CUDA_VISIBLE_DEVICES=X,Y
```

#### 问题3：训练时间过长
**症状**: 训练超过预期时间
**可能原因**:
- vLLM 初始化慢
- 梯度累积步数过大

**应对策略**:
- 减少 `gradient_accumulation_steps`（默认4）
- 使用更小的 `max_length`
